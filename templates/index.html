About This System

This Decision Support System helps you select the best Steam games based on multiple criteria using advanced mathematical methods:

AHP (Analytical Hierarchy Process): Determines the importance weight of each criterion through pairwise comparison.

SAW (Simple Additive Weighting): Ranks game alternatives based on the weighted criteria from AHP.

The criteria analyzed in this system include:

  • Price
  • Recommendations Total
  • Release Year
  • RAM Minimum
  • CPU Minimum
  • GPU Minimum
  • Rating Steam
  • Steam Chart Player 30d Avg
  • Steam Chart Current Player
